## ðŸ“„ Project Overview
**SRIMS** is an **Intelligent Decision Support System (IDSS)** designed to assist retail managers in making data-driven decisions. The system analyzes historical sales data to predict inventory needs, segment customers, and analyze market trends using advanced Artificial Intelligence and Machine Learning techniques.

This project was developed as a requirement for the **Decision Support Systems** course.
